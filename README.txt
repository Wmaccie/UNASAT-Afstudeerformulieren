UNASAT Afstudeerformulieren webapp

Bestandenstructuur:
- index.html
- assets/unasat-logo.png  (zelf toevoegen)
- templates/AB_afstudeerbegeleider/*.pdf
- templates/BB_bedrijfsbegeleider_complete/*.pdf

Publiceren:
1. Upload de volledige map naar GitHub.
2. Zorg dat index.html in de root staat.
3. Zet GitHub Pages aan.
4. Open de GitHub Pages URL, niet rechtstreeks het lokale file:// bestand.

PDF-coördinaten:
De app gebruikt startcoördinaten in het forms-object in index.html.
Na de eerste test-export kun je x/y/width/height per veld fine-tunen.
